# memory
The subdomain for any memory training tools I create.

## For Development
Get the repo
```zsh
git clone https://github.com/edamamev/memory`
```

Install packages
```sh
npm install
```

Compile and Hot-Reload for Development
```sh
npm run dev
```

Compile and Minify for Production

```sh
npm run build
```

Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
